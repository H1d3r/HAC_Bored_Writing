# hjt_fileread

会捷通云视讯存在任意文件读取漏洞


## 工具利用

python3 filedownload_fileread.py -u http://127.0.0.1:1111 单个url测试

python3 filedownload_fileread.py -f url.txt 批量getshell

python3 filedownload_fileread.py -c http://127.0.0.1:1111 进入cmd
